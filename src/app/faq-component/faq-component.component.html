Where is data coming from?

We are using public APIs for updating the website. APIs are provided by rapidapi.com for getting information about cases in world and api.covid19india.org for cases in India.

What resources are we using to publish the data?
UI Code is open to all. It can be accessed from GITHUB. If you do not want to set it up in local feel free to use Stackblitz to play around. A Big Thanks to SURGE.SH to publish our web app.
Is this website for commercial use?
No. This website is merely for educational purposes and awareness.
